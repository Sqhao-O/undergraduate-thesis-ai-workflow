### 2. Pandoc — 文档转换工具

| 项目信息 | 内容 |
|---------|------|
| **名称** | Pandoc |
| **作者** | [John MacFarlane](https://github.com/jgm) |
| **GitHub** | https://github.com/jgm/pandoc |
| **官网** | https://pandoc.org/ |
| **功能** | 通用文档转换器，支持 Markdown 转换为 DOCX、PDF、HTML 等数十种格式 |
| **许可证** | GPL-2.0-or-later |

> Pandoc 是开源社区最重要的文档转换工具之一，被广泛应用于学术出版和技术写作领域。
